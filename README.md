 A unified KPI framework is essential for decision support. Without consistent definitions, 
different teams can compute the same metric in different ways and reach conflicting conclusions. 
To avoid that, a KPI catalog should define each metric, its formula, refresh frequency, and 
source fields. In this report, KPIs such as Net Sales, Profit, Profit Margin, Average Order Value 
(AOV), Discount Rate, Online Share, CSAT, and Return Rate are defined and used consistently 
across SQL validation and Power BI visuals.
